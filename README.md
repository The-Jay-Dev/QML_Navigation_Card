# Navigation Card

A Qt Quick demo card that explores creating visually styled UI cards using the Qt5GraphicalEffects library. This project demonstrates how layered images, icons, overlays, and visual effects can be combined to create a polished navigation-themed card interface.

## Features

- Navigation-themed card design
- Custom visual card layout
- Layered image composition
- Icon overlay styling
- Demonstrates Qt Quick UI design techniques
- Uses Qt5GraphicalEffects for enhanced visual presentation

---

## Image Sources

- Icons are from [Ionicons](https://ionic.io/ionicons) for demonstrational use
- Background images are from [Pexels](https://www.pexels.com/) for demonstrational use

---

## Notes

Due to limitations of the graphical effects library, this demo includes pre-edited images to demonstrate the benefits of stacking visual elements with components from the library.

---

## Example Use Cases

- Qt Quick UI design
- Learning card-based interface layouts
- Experimenting with graphical effects
- Building reusable visual components
- Demonstrating image layering in QML
